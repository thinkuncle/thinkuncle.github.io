Vim的命令区分大小写!大部分命令能和数字组合使用，此时数字在命令前面,或是两个命令之间。

##符号约定
当本文涉及组合键的时候:
- 如果是加号+，表示两个键先后按下（有时也省略加号）；
- 如果是减号-，表示两个键同时按下。
- 本文组合键中间无空格。
